## Installation

```sh
npm install @peerigon/fractals-typescript  --save
```

## Usage

### Adding numbers

```javascript
import { add } from "@peerigon/fractals-typescript";

add(1, 2); // 3
```

## API Reference

### `add()`

Adds two numbers.

```javascript
import { add } from "@peerigon/fractals-typescript";

add(1, 2); // 3
```

**Parameters**:

- `a` (number): The first number
- `b` (number): The second number

**Returns**: `number` - Result of the addition
